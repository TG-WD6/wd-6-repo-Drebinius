Box models

Opdracht 1

er zijn 2 verschillende soorten box, block en inline. deze worden toegepast op outer display type


block gedraagt zich als volgend

  • begint op een nieuwe lijn

  • de box wordt vaak zo breed als de regel waarop hij staat

  • kan gestyled worden met height en width

  • padding, margin en border duwen andere elementen uit de weg

sommige HTML elementen zoals <h1> en <p> hebben standaard een block box als outer display type

inline gedraagt zich als volgend

  • begint op niet een nieuwe lijn

  • height en width beinvloeden de box niet

  • vertical padding, margins en borders zijn van toepassing,
    maar hebben geen invloed op andere inline boxes

  • horizontal padding, margin en border zijn van toepassing,
    en zijn wel van invloed op andere inline boxes

Opdracht 4

  • Content
    • Content is de inhoud van het element waar het block aan gelinkt staat. Zoals deze tekst de inhoud van deze paragraph is.

  • Padding
    • Padding is de volgende 4 zijdige 'laag' om de contentbox heen. Hier staat ook de background color aan vast.

  • Border
    • Border is de following 4 zijdige layer om de padding heen

  • Margin
    • margin is de 4 zijde layer 'whitespace'. wordt vooral gebruikt voor positionering en om meerdere boxes op elkaar af te stellen.
    als 2 naastgelegen boxes beiden een margin value hebben, word alleen de grootste margin value grbruikt
